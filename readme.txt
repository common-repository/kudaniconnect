== Description ==

KudaniCloud is the leading content marketing platform for small businesses.
You can create and curate interesting content that will increase your traffic and leads.

KudaniConnect helps users connect KudaniCloud to their wordPress blog with ease.
Using KudaniCloud you can:

*   Plan your content using inbuilt editorial calendar
*   Import all your blog feed subscriptions
*   Create dynamic images that get attention
*   Generate prevent headlines to get clicks
*   Automatically promote yours and other peoples content to social media
*   Built in workflow and authorisation is perfect for teams.

learn more at: <a href="https://www.kudani.com/?utm_source=wordpressorg&utm_medium=description&utm_content=plugin&utm_campaign=version1">www.kudani.com</a>

== Installation ==

1. Sign in to your WordPress blog 
2. Go to Plugins and click 'Add New'
3. Search for KudaniConnect
4. Activate the plugin through the 'Plugins' menu in WordPress
